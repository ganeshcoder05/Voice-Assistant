# 🎙️ Voice Assistant

A simple and interactive Voice Assistant built using Python. It listens to voice commands, processes them, and responds with voice output.

## 📌 Features

- 🎤 Voice Command Recognition
- 🔊 Text-to-Speech Response
- ⚡ Fast Command Processing
- 🖥️ User-Friendly Interface
- 🔄 Continuous Listening Mode
- 🛑 Exit Command Support

---

## 🛠️ Technologies Used

- Python
- SpeechRecognition
- Pyttsx3
- PyAudio
- OS Module

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ganeshcoder05/Voice_Assistance.git
```

Move to project folder:

```bash
cd Voice_Assistance
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python f.py
```
